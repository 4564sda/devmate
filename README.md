\# DevMate



DevMate 是一个面向开发者的智能编程 Agent 项目，用于帮助生成和修改代码库。



\## 当前进度



\### 第一阶段：环境与基础

当前已完成：

\- 使用 `uv` 管理项目环境

\- 使用 Python `3.13`

\- 使用 `pyproject.toml` 管理依赖

\- 创建 `config.toml` / `config.toml.example`

\- 规划 LangChain + MCP + RAG 的整体实现路径



\## 技术路线



\- Python 3.13

\- uv

\- LangChain

\- OpenAI Compatible API

\- Tavily

\- MCP (Streamable HTTP)

\- RAG

\- LangSmith

\- Docker
